# 🤖 StatBot Pro – AI Data Analyst Tool

StatBot Pro is an AI-powered data analysis tool that allows users to interact with CSV data using natural language. It can analyze datasets, generate insights, and create charts automatically.

This project is built using Python, Pandas, Matplotlib, and LangChain with a local LLM (Ollama).

---

## 🚀 Features

- 📊 Ask questions about CSV using natural language  
- 📈 Automatic data analysis  
- 📉 Generate charts (Line, Bar, Pie)  
- 🧠 AI-powered responses using LLM  
- ⚡ CLI-based interactive tool  
- 📁 Charts saved automatically in folder

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

git clone https://github.com/niru7400/statbot-pro-ai-data-analysis-2026.git

### 2️⃣ Navigate to Project

cd statbot-pro-ai-data-analysis-2026

### 3️⃣ Install Requirements

pip install -r requirements.txt

### 4️⃣ Install Ollama (if not installed)

Download from:  
https://ollama.com/download

Run model:

ollama run llama3

---

## ▶️ Run the Project

python app.py
